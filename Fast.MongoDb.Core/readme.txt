
    mongodb表是文档,是json

    show collections显示所有的表

    db.serverStatus() 查询mongo信息

    _id是mongo的主键唯一id,插入时不用赋值，因为赋值了会查找这个是不是唯一，在所有里的数据库里找。

    db.表名.find({'字段名1':值1}，{'显示字段名1':''}...) // 如果字段类型是数字类型，与字符类型，这里写值要注意，字符类型必须要引号

    db.表名.find({"字段名": {$elemMatch:{'字段名1':值1}}},{'显示列名':'','显示列名2':'','显示列名3':''})//json 字段下有list查询方法， 查询匹配文档

    db.表名.find({"字段名": {条件:值1,条件:值2}},{'显示列名':'','显示列名2':'','显示列名3':''}})

    db.表名.find({"字段名": {条件:值1,条件:值2}},{'显示列名':'','显示列名2':'','显示列名3':''}}).sort({'字段名':-1});

    db.表名.update({'字段名1':{条件:值}},{$set:{'字段名1',值}})

    db.表名.updateMany({'字段名1':{条件:值}},{$set:{'字段名1',值}})

    db.表名.delete({'字段名1':{条件:值}})

    db.表名.deleteMany({'字段名1':{条件:值}})

    条件

    $lt 小于

    $lte 小于等于

    $gt 大于

    $gte 大于等于

    $ne 不等于

    $in 包含

    $nin 不包含

    $or 或者

    $exists 存在

    limit(数量) 取几条

    skip(数量) 向下几个

    sort({'字段名':1}) 排序 1：升序，-1：降序
